## üß† Skills Demonstrated

* Writing basic and advanced `SELECT` queries
* Filtering data using `WHERE`, `LIKE`, and `BETWEEN`
* Sorting results with `ORDER BY`
* Removing duplicates using `DISTINCT`
* Using aggregate functions such as `COUNT` and `AVG`
* Working with **subqueries**
* Limiting and ranking results
* Performing **population and demographic analysis**
* Joining multiple tables using `JOIN`
